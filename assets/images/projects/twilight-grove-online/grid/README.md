# Twilight Grove Online screenshots

Drop four screenshots in this folder for the project page grid.

Suggested filenames:

1. 01-town-square.webp
2. 02-inventory.webp
3. 03-quest.webp
4. 04-cross-platform.webp

Any image files with these extensions will be picked up automatically by the grid include: .png, .jpg, .jpeg, .webp, .gif, .avif.
